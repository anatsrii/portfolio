# Portfolio React 🖥️

A desktop-inspired portfolio website built with React + Vite. Features a Windows-like interface with draggable windows, taskbar, and bilingual support (Thai/English).

![Portfolio Screenshot](./screenshot.png)

## ✨ Features

- 🪟 **Desktop Environment** - Windows-style interface with minimize/maximize/close
- 📌 **Taskbar** - Toggle windows from the taskbar
- 🌐 **Bilingual** - Thai/English language switcher
- ⏰ **Realtime Clock** - Live clock in taskbar
- 🎨 **Pixel Art Theme** - Retro desktop aesthetic
- 📱 **Responsive** - Works on desktop and mobile

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite 6
- **State Management:** Redux Toolkit
- **Styling:** CSS3 with custom fonts
- **i18n:** react-i18next
- **Icons:** Custom pixel art icons

## 🚀 Getting Started

### Prerequisites
- Node.js or Bun

### Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd portfolio-react

# Install dependencies
bun install

# Run development server
bun dev
```

Open http://localhost:5173/ to view in browser.

### Build for Production

```bash
bun run build
```
